# Cannabis Label Text Helper — Legacy Archive

This project is an archived/reconstructed version of a lightweight internal workflow tool I built around 2017 during the Nevada medical cannabis era.

The goal was simple: reduce inconsistent manual data entry when copying COA/compliance details into a single free-text label field in MJFreeway.

This repo is preserved as a product/workflow case study, not as a modern production tool.

## Problem

Inventory/compliance staff had to manually enter COA and label details into a large free-text field. Formatting varied by person, which created inconsistent labels and avoidable corrections.

## Solution

A simple browser-based helper normalized repeated inputs into a consistent copy/paste output block.

## User Feedback

The first version included a typing speed timer/high-score mechanic. After seeing that it stressed out a slower typist, I removed the timer and prioritized accuracy and comfort over speed.

## Product Lessons

- Standardize repeated workflows
- Reduce user input variation
- Build guardrails around compliance tasks
- Watch real users, then iterate
- Fun features are not valuable if they increase stress
